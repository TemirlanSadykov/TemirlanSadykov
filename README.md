### 🧾 Kakei - LINE Budget Tracker

**Kakei** is a bilingual Japanese/English LINE chatbot that helps users track personal spending by scanning receipt images, saving transactions, and showing monthly spending breakdowns inside a LIFF miniapp.

The project combines chatbot UX, receipt parsing, structured transaction storage, user session management, and an in-LINE dashboard experience.

**Tech Stack:** Python, FastAPI, PostgreSQL, Redis, LINE Messaging API, LINE Login, LIFF, Gemini 2.5 Flash, async HTTP

**Highlights:**

- Built a bilingual LINE chatbot with Japanese and English language support
- Implemented receipt image processing with AI-powered structured extraction
- Parsed receipt data into transaction fields such as store, amount, category, date, and payment method
- Created a confirmation and edit flow with step-by-step user interaction
- Stored transactions in PostgreSQL
- Used Redis-backed session state with TTL for conversation flow management
- Built a LIFF miniapp showing monthly spending statistics by category
- Added payment method filtering, month navigation, category breakdowns, transaction expansion, and delete actions
- Verified LINE access tokens for authenticated LIFF API access
- Implemented webhook signature verification for secure LINE event handling
